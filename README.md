🏢 Gestão Instituição
O Gestão Instituição é uma API REST desenvolvida para centralizar e otimizar a administração de processos institucionais. O foco do projeto é oferecer uma estrutura robusta para o gerenciamento de entidades, usuários e fluxos de trabalho internos, garantindo integridade de dados e escalabilidade.

🚀 Tecnologias Utilizadas
Este projeto utiliza o que há de mais moderno no ecossistema Java:

Java 17: Linguagem principal.

Spring Boot 3: Framework para agilizar o desenvolvimento.

Spring Data JPA: Abstração de persistência de dados.

Hibernate: Implementação do ORM.

Maven: Gerenciador de dependências e build.

H2 Database / PostgreSQL: Banco de dados (desenvolvimento).

Bean Validation: Para validação rigorosa de dados de entrada.

[x] Arquitetura Base: Configuração inicial do Spring Boot e pacotes.
